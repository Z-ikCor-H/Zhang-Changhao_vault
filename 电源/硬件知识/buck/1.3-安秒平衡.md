# 第一章 Buck电路 第三节：安秒（电荷）平衡

[[1.2-伏秒平衡|伏秒平衡]]给出了稳态增益。本节讨论另一个基本稳态条件——安秒平衡（Amp-Second Balance，电容进入稳态的标志）：**电感电流的纹波分量流入输出电容后，会在 $U_o$ 上产生多大的电压纹波？电容在频域中的行为如何理解？** 通过安秒平衡定量计算输出纹波，给出电容选型的第一依据，并从s域/相量分析电容特性。

---

## 一、安秒平衡的引出

### 1.1 问题

电感电流存在纹波（三角波波动），这个纹波电流流到哪里去了？它必然会在输出电压上产生纹波。许多开关电源对纹波有严格要求（如 <5%、<1% 等），因此需要定量估算输出纹波的大小。
[[1.2-伏秒平衡|式(1.2-1)]]
### 1.2 电感电流的分解

将电感电流 $i_L$ 分解为两个部分：

$$i_L = I_L + \Delta i_L$$

- **$I_L$（直流分量）**：电感电流的平均值
- **Δ$i_L$（交流分量）**：电感电流的纹波（峰峰值三角波）
![[1.3-1.png|397]]
由于负载电流 $I_o$ 几乎不变（输出电压 $U_o$ 恒定，负载 R 不变），故：
- **交流分量 Δ$i_L$ 全部流入输出电容 $C_o$**（电容对交流呈现低阻抗，$i_{Co} = \Delta i_L$）
- **直流分量 $I_L$ 全部流入负载**

因此有：

$$I_L = \bar{i}_L = I_o$$

即：**Buck电路中，电感电流的平均值 = 负载电流。**

> 证明方法：在输出节点处由 KCL，$i_L = i_{Co} + I_o$。取一个周期平均值，电容电流平均值 = 0（稳态），故 $\bar{i}_L = I_o$。

## 二、电容电流波形与充放电行为

### 2.1 电容电流波形

电感电流 $i_L$ 是带有直流偏置的三角波。从中减去直流分量 $I_L$(=I_o) 后，得到电容电流 $i_{Co}$：

$$i_{Co} = i_L - I_L = i_L - I_o$$

$i_{Co}$ 是一个**对称的三角波**，以零为中心上下波动：
- 正峰值：$+\frac{\Delta i_L}{2}$
- 负峰值：$-\frac{\Delta i_L}{2}$

### 2.2 三个关键时间点

以 $i_L$ 与平均值 $I_L$ 的交点作为分界，一个周期可划分为三个关键时间点：
![[1.3-2.png|296]]

| 时间点 | 位置 | 物理含义 |
|---|---|---|
| $t_1$ | $t_1 = \frac{t_{on}}{2}$（导通段中点） | $i_L$ 从 < $I_L$ 变为 > $I_L$，电容电流从负变正过零 |
| $t_2$ | $t_2 = t_{on} + \frac{t_{off}}{2}$（关断段中点） | $i_L$ 从 > $I_L$ 变为 < $I_L$，电容电流从正变负过零 |
| $t_3$ | $t_3 = T$（周期结束点） | = 下一周期的 $t_1$（$t_3$ = T + $t_1$） |

> 理解方法：三角波中 $I_L$ 是平均值线。把三角波"多出"平均值的部分（以 $I_L$ 为轴旋转180°）恰好能填补"低于"平均值的缺口，因此交点就是各段的**中点**。$I_L$ 线相当于大三角形的**中位线**。

### 2.3 分阶段充放电行为

| 时间段       | $i_L$ vs $I_L$ | $i_{Co}$ 方向 | 电容状态   | 物理过程                     |
| --------- | ---------- | ------- | ------ | ------------------------ |
| $t_1$ 之前    | $i_L$ < $I_L$  | 负       | **放电** | 电感提供的电流不足，电容 + 电感共同给负载供电 |
| $t_1$ ~$t_2$  | $i_L$ > $I_L$  | 正       | **充电** | 电感电流大于负载需求，同时给负载供电和给电容充电 |
| $t_2$ ~ $t_3$ | $i_L$ < $I_L$  | 负       | **放电** | 电感电流再次不足，电容再次放电补充负载电流    |

充放电时间各占半个周期：

$$t_2 - t_1 = t_3 - t_2 = \frac{t_{on}}{2} + \frac{t_{off}}{2} = \frac{T}{2}$$

## 三、安秒平衡（Amp-second Balance）

### 3.1 原理

在稳态下，电容电压在一个周期内不能有净变化 → **电容的充电电荷 = 放电电荷**。

$$\Delta Q_{in} = \Delta Q_{out}$$

电荷 = 电流 × 时间，单位为**安·秒**。因此在稳态下电容达到**安秒平衡**。稳态下电容电压平均值恒定，维持 $U_o$ 不变。

> 安秒平衡是电容进入稳态的标志性事件，与电感的[[1.2-伏秒平衡|伏秒平衡]]并列为开关电源的两大基本稳态条件。

### 3.2 充电电荷计算

充电电荷 = 电容电流正半周三角形的面积（$i_{Co}$ 波形与时间轴所围成的红色三角形面积）：

$$\Delta Q_{in} = \int_{t_1}^{t_2} i_{Co}(t) \, dt = \text{三角形面积} = \frac{1}{2} \times \text{底} \times \text{高}$$

- 底（充电时间）= $t_2 - t_1 = \frac{T}{2}$
- 高（电容电流峰值）= $\frac{\Delta i_L}{2}$

$$\boxed{\Delta Q_{in} = \frac{1}{2} \times \frac{T}{2} \times \frac{\Delta i_L}{2} = \frac{T \cdot \Delta i_L}{8}}$$

放电电荷的黄色三角形面积与之相等。

## 四、输出电压纹波

### 4.1 基本关系

$$U_C = \frac{Q}{C} \quad \Rightarrow \quad \Delta u_o = \frac{\Delta Q_{in}}{C_o}$$

代入 ΔQ：

$$\boxed{\Delta u_o = \frac{T \cdot \Delta i_L}{8 C_o}}$$

这就是**Buck电路输出电压纹波峰峰值的标准公式**。

### 4.2 电容电压与电流的相位关系

时域关系：

$$u_C = \frac{1}{C_o} \int i_{Co}(t) \, dt$$

复频域（s域/相量）关系：

$$U_{Co}(s) = \frac{I_{Co}(s)}{s C_o}, \quad s = j\omega \quad\Rightarrow\quad U_{Co} = \frac{I_{Co}}{j\omega C_o} = -j\,\frac{I_{Co}}{\omega C_o}$$

> 拉式变换时域积分性质
> [信号与系统_第二版_化学工业出版社, 页面 232]

$$\begin{aligned}
&\qquad\text{若}\quad &&\boxed{x(t) \stackrel{\mathcal{L}}{\longleftrightarrow} X(s), \quad \text{ROC}=R} \\[6pt]
&\text{则}\quad &&\boxed{\int_{-\infty}^{t} x(\tau) \, d\tau \stackrel{\mathcal{L}}{\longleftrightarrow} \frac{1}{s}X(s), \quad \text{ROC 包含 } R \cap \{\text{Re}\{s\} > 0\}}
\end{aligned}$$

-j 表示顺时针旋转90°（即滞后90°），因此：

> **电容电压滞后电容电流90°。** 电容电压具有惯性，不能突变。

这意味着：
- 当电容电流**过零**时（$i_{Co}$ = 0）→ 电容电压达到**峰值**（最大或最小）
- 当电容电流**达到峰值**时 → 电容电压恰好**过零**

### 4.3 纹波波形具体对应

在右侧配套的完整时序波形中：
- **$t_1$ 时刻**：电容电流过零（由负变正），输出电压 $u_o$ 处于**谷值**（最小值）
- **$t_1$ ~ $t_2$**：电容充电，输出电压持续上升
- **$t_2$ 时刻**：电容电流过零（由正变负），输出电压 $u_o$ 处于**峰值**（最大值）
- **$t_2$ ~ $t_3$**：电容放电，输出电压持续下降
- **$t_3$ 时刻**：= 下一周期的 $t_1$，输出电压回到谷值

## 六、本节小结

| 主题 | 核心结论 |
|---|---|
| 安秒平衡 | 稳态下电容电流平均值为零（或电容电荷净变化量为零） |
| 电流分配 | 电感电流直流分量 $I_L$ → 流入负载；交流分量 $\Delta i_L$ → 流入输出电容 |
| 输出电压纹波 | $\Delta U_o = \dfrac{T \cdot \Delta i_L}{8C}$，由电容充放电产生 |
| 电容频域特性 | 电容电压滞后电流90°，具有惯性、不能突变 |
