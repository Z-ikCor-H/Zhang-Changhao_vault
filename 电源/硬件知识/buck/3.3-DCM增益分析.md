# 第三章 Buck DCM原理 第三节：DCM增益分析

[[3.2-DCM工作过程|DCM模式]]下的增益不等于 $U_o/U_{in}=D$。本节通过[[1.2-伏秒平衡|伏秒平衡]]和[[1.3-安秒平衡|安秒平衡]]联合推导DCM增益：**增益与哪些参数有关？为什么轻载时输出电压偏高？如何证明DCM增益大于CCM增益？** 给出增益公式及严格证明。

---

## 一、推导思路

DCM模式下的增益 $U_o/U_{in}$ 不再简单地等于占空比 $D$，还与电感 $L$、负载 $R_0$、开关周期 $T$ 有关。

推导分两步：
1. **伏秒平衡** → 用 $D$ 和 $D_1$ 表示增益
2. **安秒平衡（电感电流平均值 = 负载电流）** → 求出 $D_1$
3. 代入 → 得到增益的完整表达式

---

## 二、伏秒平衡

DCM模式下电感两端电压波形：

![[3.3-1.png|387]]

- **模态一（$DT$）：** $U_L = U_{in} - U_o$
- **模态二（$D_1 T$）：** $U_L = -U_o$
- **模态三：** $U_L = 0$（电感电流为零且不变）

由伏秒平衡（电感电压平均值为零）：

$$(U_{in} - U_o) \cdot D \cdot T = U_o \cdot D_1 \cdot T$$

整理得：

$$\boxed{\frac{U_o}{U_{in}} = \frac{D}{D + D_1}}$$

> **与CCM对比：** CCM中 $D_1 = 1-D$，代入得 $U_o/U_{in} = D$。DCM中 $D_1 < 1-D$（因为 $T_D > 0$），因此增益表达式不同。

关键问题：**$D_1$ 是多少？** 需要借助电感电流的平均值来求解。

---

## 三、用电感电流平均值求 $D_1$

### 3.1 电感电流平均值

在Buck电路中，电感与负载串联，电感电流平均值等于负载电流：

$$I_L = I_o = \frac{U_o}{R_0}$$

### 3.2 三角形面积法

DCM模式下电感电流为三角形（高为 $I_{LP}$，底边为 $(D+D_1)T$）：

![[3.3-1.png|384]]

$$I_L = \frac{1}{2} \cdot I_{LP} \cdot (D + D_1)$$

### 3.3 用关断阶段表示 $I_{LP}$

在模态二（二极管导通，电感放电）中：

$$I_{LP} = \frac{U_o}{L} \cdot D_1 \cdot T$$

> 也可用模态一表示：$I_{LP} = \frac{U_{in} - U_o}{L} \cdot D \cdot T$，本质等价。

### 3.4 联立求解

将 $I_{LP}$ 代入 $I_L$ 表达式：

$$\frac{U_o}{R_0} = \frac{1}{2} \cdot \frac{U_o}{L} \cdot D_1 T \cdot (D + D_1)$$

$U_o$ 约去（假设 $U_o \neq 0$），整理得：

$$D_1(D + D_1) = \frac{2L}{R_0 T}$$

即：

$$\boxed{D_1^2 + D \cdot D_1 - \frac{2L}{R_0 T} = 0}$$

这是关于 $D_1$ 的一元二次方程。

---

## 四、求解 $D_1$

由求根公式 $D_1 = \frac{-D \pm \sqrt{D^2 + 8L/(R_0 T)}}{2}$：

- 取负号 → $D_1 < 0$（舍去，物理上 $D_1$ 必须为正）
- 取正号 → 得到物理有意义的解

$$\boxed{D_1 = \frac{-D + \sqrt{D^2 + \frac{8L}{R_0 T}}}{2}}$$

---

## 五、DCM增益表达式

将 $D_1$ 代入 $U_o/U_{in} = D/(D + D_1)$：

$$\frac{U_o}{U_{in}} = \frac{D}{D + \frac{-D + \sqrt{D^2 + \frac{8L}{R_0T}}}{2}}$$

分子分母同乘2：

$$\frac{U_o}{U_{in}} = \frac{2D}{2D - D + \sqrt{D^2 + \frac{8L}{R_0T}}}$$

最终得到DCM增益公式：

$$\boxed{\frac{U_o}{U_{in}} = \frac{2D}{D + \sqrt{D^2 + \frac{8L}{R_0 T}}}}$$

### 5.1 公式解读

DCM增益取决于四个参数：$D$、$L$、$R_0$、$T$（或 $f_s$），而非CCM中单纯的 $D$。

令无量纲参数：
$$\tau = \frac{L}{R_0 T} = \frac{L \cdot f_s}{R_0}$$

则增益可简写为：
$$\frac{U_o}{U_{in}} = \frac{2D}{D + \sqrt{D^2 + 8\tau}}$$

---

## 六、负载对增益的影响

从增益公式可以看出：

- $R_0 \uparrow$（负载减轻） → $\tau = L/(R_0 T) \downarrow$ → 分母中根号项减小 → **增益 $\uparrow$**
- $R_0 \downarrow$（负载加重） → $\tau \uparrow$ → 分母中根号项增大 → **增益 $\downarrow$**

> **物理意义：** DCM模式下，电感在每个周期存储的能量全部释放给负载。若负载消耗变少（$P_o \downarrow$），多余的电荷会使输出电压上升（增益增大）。为维持 $U_o$ 不变，必须减小占空比 $D$ 来降低电感储能。

### 物理过程解释

1. 电感储能峰值：$E_L = \frac{1}{2} L \cdot I_{LP}^2$
2. 每个周期，电感将所有储能释放到输出端
3. 若 $P_o$（负载消耗）减少 → 输出电容电压上升 → $U_o$ 上升
4. 闭环调节 → 减小 $D$ → $I_{LP}$ 降低 → 电感储能减少 → $U_o$ 回落

这就是DCM模式下负载变化会影响增益的根本原因。

---

## 七、DCM增益与CCM增益的比较

### 7.1 结论

> **在相同占空比 $D$ 下，DCM模式的增益大于CCM模式的增益。**

即：
$$\left.\frac{U_o}{U_{in}}\right|_{DCM} > \left.\frac{U_o}{U_{in}}\right|_{CCM} = D$$

### 7.2 证明

当 $L < L_{crit}$（进入DCM的前提），将 $L_{crit} = \dfrac{R_0(1-D)T}{2}$ 代入：

$$L < \frac{R_0(1-D)T}{2} \quad \Rightarrow \quad \frac{8L}{R_0T} < 4(1-D)$$

在增益公式中，我们需要证明：

$$\frac{2D}{D + \sqrt{D^2 + \frac{8L}{R_0T}}} > D$$

即等价于证明：

$$\frac{2}{D + \sqrt{D^2 + \frac{8L}{R_0T}}} > 1$$

$$D + \sqrt{D^2 + \frac{8L}{R_0T}} < 2$$

用 $L_{crit}$ 替代 $L$（此举会使根号项增大，是放缩方向）：

$$D + \sqrt{D^2 + 4(1-D)} < D + \sqrt{D^2 - 4D + 4}$$

$$= D + \sqrt{(2-D)^2}$$

由于 $0 < D < 1$，$\sqrt{(2-D)^2} = 2-D$：

$$D + (2-D) = 2$$

因此：
$$D + \sqrt{D^2 + \frac{8L}{R_0T}} < 2$$

证毕。

### 7.3 实际含义

- 若闭环控制在DCM模式下仍维持CCM的占空比 → 输出电压会**偏高**
- 实际变换器中，反馈环路会自动减小占空比来维持目标 $U_o$
- 如果变换器设计的电感 $L > L_{crit}$，在负载减轻到使 $R_0 > R_{crit}$ 时仍会自动进入DCM → **任何CCM设计的变换器都不能保证全负载范围CCM**

---

## 八、DCM增益曲线特征

| 参数变化 | $\tau = L/(R_0T)$ | 增益 | 说明 |
|---|---|---|---|
| $R_0 \uparrow$（轻载） | $\downarrow$ | $\uparrow$ | 负载越轻，增益越高 |
| $R_0 \downarrow$（重载） | $\uparrow$ | $\downarrow \to D$ | 重载时趋近CCM增益 |
| $L \downarrow$ | $\downarrow$ | $\uparrow$ | 电感越小，DCM效应越强 |
| $L \uparrow \to L_{crit}$ | $\uparrow$ | $\downarrow \to D$ | 趋近临界点，增益回到 $D$ |

> **极限验证：** 将BCM条件 $L = \dfrac{R_0(1-D)T}{2}$ 代入DCM增益公式 → 化简得 $U_o/U_{in} = D$，与CCM公式一致。这说明BCM确实是CCM和DCM的**无缝衔接点**。

---

## 九、本节小结

| 主题 | 核心结论 |
|---|---|
| 伏秒平衡 | $\dfrac{U_o}{U_{in}} = \dfrac{D}{D + D_1}$ |
| $D_1$ 求解 | 由 $I_L = I_o$ 和 $I_{LP} = \frac{U_o}{L}D_1T$ 导出二次方程 |
| 增益公式 | $\dfrac{U_o}{U_{in}} = \dfrac{2D}{D + \sqrt{D^2 + \frac{8L}{R_0T}}}$ |
| 增益与负载关系 | $R_0 \uparrow$ → 增益 $\uparrow$（轻载时输出电压升高） |
| DCM vs CCM增益 | DCM增益 **>** CCM增益（相同 $D$ 下），已严格证明 |
| 衔接性 | BCM条件代入DCM公式 → 退化为 $U_o/U_{in} = D$ |
