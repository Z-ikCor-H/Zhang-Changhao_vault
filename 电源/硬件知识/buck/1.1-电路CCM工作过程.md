# 第一章 Buck电路 第一节：CCM工作过程

## 一、Buck电路拓扑简介

典型的Buck电路由以下元件组成：
- **开关管（MOS管）Q**
- **二极管 D**（续流二极管）
- **电感 L**
- **输出电容 C**
- **负载 R**

Buck电路是最基本的DC-DC变换器拓扑之一。半桥、全桥、推挽、正激、反激等拓扑都可以看作Buck电路的衍生或组合，因此学习Buck电路是学习开关电源的基础。

---

## 二、理想化假设

为了便于原理分析，做出以下假设：

1. **所有功率器件均为理想器件**（MOS管、二极管效率100%）
   - MOS管：无结电容，开关转换时间可视为瞬时完成，导通电阻 R_DS(on) = 0Ω
   - 二极管：导通无压降，无反向恢复
2. **电感和电容均为无损储能元件**
3. **输出电容足够大**，输出电压 U_o 无波动，视为恒定值

---

## 三、电感公式的理解

核心公式：

$$U_L = L \cdot \frac{di_L}{dt}$$

### 3.1 物理层面理解

电感两端电压与**电感电流的变化率**成正比，与电感电流的**绝对值大小无关**。

- 即使电感电流达到1000A，只要不变化（直流），电感两端电压就是 **0V**
- 即使电感电流为0A，只要在变化，电感两端就会有电压

> ==这是初学者容易混淆的关键点：电感电压不由电流大小决定，而由电流变化快慢决定。==

### 3.2 数学层面理解

该公式是一个**微分方程**。电感电流 i_L(t) 对时间 t 可导，说明：

- **i_L(t) 必须是连续函数**（可导必连续）
- 电容电压可以突变，但**电感电流不能突变**

因此可以出现的波形：
- 正弦波 ✓（连续且可导）
- 三角波 ✓（连续，尖点处左右导数不等但函数仍连续）
- 方波 ✗（存在间断点，不连续）

如果电感电流出现跳变（方波），则 di_L/dt → ∞，意味着两端产生无穷大电压——这在物理上不可能实现，会导致空气电离产生电弧。这就是为什么开关断开大电流时会产生电弧的原因：导线中存在寄生电感，电流突然被切断时电感感应出极高电压，击穿空气形成电弧。

### 3.3 频域理解——s域/相量分析

对公式两边取拉普拉斯变换：

$$U_L(s) = sL \cdot I_L(s)$$

令 s = jω，得到相量形式：

$$U_L = j\omega L \cdot I_L$$

**j 的物理意义：代表逆时针旋转90°。**

- 乘以 j → 逆时针旋转90°（超前90°）              
- 乘以 j² = -1 → 旋转180°
- 乘以 j³ = -j → 旋转270°

因此：**电感电压超前电流90°**（电压领先电流90°），反过来即**电感电流滞后电压90°**。

这也解释了为什么电感电流具有"惯性"——电流不能突变，正是因为电感电压超前电流、电感对电流变化产生阻碍。

## 四、Buck电路CCM模式工作过程

Buck电路只有一个开关管，因此只有两种工作状态：

### 4.1 状态一：开关管 Q 导通（t_on）

- 开关管导通，输入电压 U_in 加到电感左端
- 二极管反向截止（承受反向电压 U_in）
- 电感两端电压：$U_L = U_{in} - U_o > 0$（Buck电路中 U_in > U_o）
- 由 $U_L = L \cdot di_L/dt$，U_L > 0 → di_L/dt > 0 → **电感电流线性上升**

电感电流变化量：

$$\Delta i_L^{(+)} = \frac{U_{in} - U_o}{L} \cdot t_{on}$$

此阶段：
- 电源 U_in 通过电感**对负载供电**
- 电感电流增大，电感储能 $E = \frac{1}{2}Li_L^2$ 增加（**电感储能**）
- 开关管电流 = 电感电流，二极管电流 = 0

### 4.2 状态二：开关管 Q 关断（t_off）

- 开关管关断，电感电流不能突变，必须寻找通路
- 二极管被迫导通（**续流二极管**的作用——为电感电流提供通路）
- 电感左端被拉到地（0V），右端为 U_o
- 电感两端电压：$U_L = 0 - U_o = -U_o < 0$
- U_L < 0 → di_L/dt < 0 → **电感电流线性下降**

电感电流变化量：

$$\Delta i_L^{(-)} = \frac{U_o}{L} \cdot t_{off}$$

此阶段：
- **电感向负载释放能量**（电感电流减小，储能减少）
- 二极管导通，承受压降（理想情况下为0V）
- 开关管承受电压 U_in

### 4.3 CCM模式定义

进入稳态后，由伏秒平衡可知：

$$\Delta i_L^{(+)} = \Delta i_L^{(-)}$$

即一个周期内电感电流净变化量为零。

**CCM（Continuous Conduction Mode，连续导通模式）：**

电感电流在一个周期内始终大于零，即 i_L 不会触碰到坐标零点。开关管再次导通时，电感电流从某个正值 i_L(min) 开始上升，不会从零开始。

> "连续"指的是电感电流始终连续、不中断到零。
