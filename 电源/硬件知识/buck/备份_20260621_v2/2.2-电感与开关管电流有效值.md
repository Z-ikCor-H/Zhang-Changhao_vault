# 第二章 Buck参数计算 第二节：电流有效值计算（一）

上一节（2.1）计算了输出电容电流的有效值，本节继续计算**电感**和**开关管**的电流有效值，为后续元器件选型（线径、损耗评估）提供依据。

---

## 一、电感电流有效值计算

### 1.1 电感电流波形

电感电流 $i_{L}$ 的波形是一个**带有直流偏置的三角波**：![[1.3-1.png|447]]

与输出电容电流（对称三角波，无直流分量）不同，电感电流中存在**直流分量 I_L**。
### 1.2 核心思路：傅里叶正交分解

利用**傅里叶分解**的思想，将电感电流分解为两个正交分量：

$$i_L(t) = \underbrace{I_L}_{\text{直流分量}} + \underbrace{i_{ac}(t)}_{\text{交流分量（三角波）}}$$

- **直流分量 $I_{L}$**：电感电流的平均值，$I_L = I_o$（输出负载电流）
- **交流分量 $i_{ac}(t)$**：对称三角波，峰峰值 $\Delta i_L$，**恰好就是流过输出电容的电流**

> **为什么可以这样分解？** 傅里叶分解将波形展开为一组**正交基**（不同频率的正弦/余弦函数）的线性叠加。不同频率的波形在数学上是**正交的**——即：
>
> $$\int_0^T \sin(n\omega t) \cdot \sin(m\omega t) \, dt = 0 \quad (m \neq n)$$
>
> 直流分量（频率为 0）与所有交流分量正交，不同频率的交流分量之间也互相正交。

### 1.3 正交性在 RMS 计算中的作用

有效值的定义：

$$I_{rms} = \sqrt{\frac{1}{T} \int_0^T i^2(t) \, dt}$$

将 $i_L(t) = I_L + i_{ac}(t)$ 代入：

$$I_{L\_rms}^2 = \frac{1}{T} \int_0^T \left[I_L^2 + 2I_L \cdot i_{ac}(t) + i_{ac}^2(t)\right] dt$$

- 第一项：$\frac{1}{T} \int_0^T I_L^2 \, dt = I_L^2$
- 第二项（交叉项）：$\frac{2I_L}{T} \int_0^T i_{ac}(t) \, dt = 0$（交流分量在一个周期内的平均值为零，这正是**正交性**的体现）
- 第三项：$\frac{1}{T} \int_0^T i_{ac}^2(t) \, dt = I_{ac\_rms}^2$

> **正交性的关键结论：交叉项积分为零 → 总 RMS 的平方 = 各分量 RMS 的平方之和。**

$$\boxed{I_{L\_rms}^2 = I_L^2 + I_{ac\_rms}^2}$$

### 1.4 代入已知结果

由 2.1 节结论，对称三角波的交流分量有效值：

$$I_{ac\_rms} = \frac{\Delta i_L}{2\sqrt{3}}$$

直流分量 $I_L = I_o$（负载电流）。

代入：

$$I_{L\_rms}^2 = I_o^2 + \left(\frac{\Delta i_L}{2\sqrt{3}}\right)^2 = I_o^2 + \frac{\Delta i_L^2}{12}$$

$$\boxed{I_{L\_rms} = \sqrt{I_o^2 + \frac{\Delta i_L^2}{12}}}$$

### 1.5 物理意义

电感的 RMS 电流由两部分构成：

| 分量                  | 来源   | 物理含义                    |
| ------------------- | ---- | ----------------------- |
| $I_o^2$             | 直流分量 | 向负载传递的有功功率              |
| $\Delta i_L^2 / 12$ | 交流纹波 | 在电感 DCR（直流电阻）上产生额外的交流损耗 |

> 该有效值后续用于**电感线径选择**（磁元件设计部分会详细讲解）：线径必须能承受该 RMS 电流的发热。

## 二、开关管（MOSFET）电流有效值计算

### 2.1 开关管电流波形

开关管仅在 $t_{on}$（导通时段）导通，其电流波形为电感电流对应时间段的**梯形波**：
![[1.3-1.png|377]]

```
i_Q (开关管电流)
^
|  I_L2  +-----------+
|        |   梯形波   |
|        |  (t_on)   |
|  I_L1  +-----------+
|        |           |
|  0     +-----------+------------------> t
|        |<--t_on--> |<-- t_off -->|
|        |<--------- T ---------->|
|
|  开关管导通时：i_Q = i_L（电感电流流过开关管）
|  开关管关断时：i_Q = 0（开关管截止）
```

- 导通时段波形为**线性上升的梯形波**（因为电感电流线性上升）
- 占空比 $D = t_{on} / T$

### 2.2 方法一：直接积分法

#### 波形表达式

在 $t_{on}$ 时段内，开关管电流线性上升：

$$i_Q(t) = I_{L1} + k \cdot t, \quad k = \frac{I_{L2} - I_{L1}}{t_{on}} = \frac{\Delta i_L}{t_{on}}$$

其中 $I_{L1}$ 为谷值，$I_{L2}$ 为峰值。

#### 有效值定义

$$I_{Q\_rms} = \sqrt{\frac{1}{T} \int_0^{t_{on}} i_Q^2(t) \, dt}$$

#### 积分计算

代入并积分（详细推导省略，利用 $\int (a+bt)^2 dt$ 标准积分公式）：

$$\begin{aligned}
\int_0^{t_{on}} i_Q^2(t) \, dt &= \int_0^{t_{on}} \left(I_{L1} + \frac{I_{L2}-I_{L1}}{t_{on}} \cdot t\right)^2 dt \\[4pt]
&= \frac{t_{on}}{3} \left(I_{L1}^2 + I_{L1} \cdot I_{L2} + I_{L2}^2\right)
\end{aligned}$$

代入 RMS 定义：

$$I_{Q\_rms} = \sqrt{\frac{1}{T} \cdot \frac{t_{on}}{3} \left(I_{L1}^2 + I_{L1} \cdot I_{L2} + I_{L2}^2\right)} = \sqrt{D \cdot \frac{I_{L1}^2 + I_{L1} \cdot I_{L2} + I_{L2}^2}{3}}$$

$$\boxed{I_{Q\_rms} = \sqrt{D \cdot \frac{I_{L1}^2 + I_{L1} \cdot I_{L2} + I_{L2}^2}{3}}}$$

#### 通用性

该公式**对任意线性化的梯形波均成立**——不论梯形波是上升、下降还是平坦，只要波形是线性函数，有效值都可以用此式计算（谷值的平方 + 峰值的平方 + 谷值×峰值，除以 3，再乘以占空比 D）。

### 2.3 方法二：直流 + 交流分解法

仿照电感电流的做法，将开关管电流分解为直流分量与交流分量：

```
i_Q 的分解
^
|  I_L2  +----------+-------
|        |   / \    |     /  ← 交流三角分量
|        |  /   \   |    /
|  I_L   |-----+----|------- ← 直流分量 I_L = (I_L1+I_L2)/2
|        |/       \ | /
|  I_L1  +----------+-------
|        |          |
|  0     +----------+------------------> t
|
|  直流分量：梯形波斜面的中点值 I_L
|  交流分量：以 I_L 为中心上下波动的三角波
```

#### 分解

- **直流分量**：$I_L = \dfrac{I_{L1} + I_{L2}}{2}$（梯形波斜面的**中值**，注意这里不是整个周期的平均值，而是导通时段的中心值）
- **交流分量**：峰值为 $\pm \Delta i_L / 2$ 的三角波

#### 利用正交性

与电感电流同样的道理，利用傅里叶正交性，直流分量与交流分量互不影响：

$$I_{Q\_rms}^2 = I_{Q\_dc}^2 + I_{Q\_ac\_rms}^2$$

但需要注意——开关管电流仅在 $D \cdot T$ 时段内有值。积分区间从 $[0, T]$ 变为 $[0, t_{on}]$：

$$I_{Q\_rms}^2 = \frac{1}{T} \int_0^{t_{on}} \left(I_L + i_{ac}(t)\right)^2 dt$$

由于正交性（交叉项积分为零），展开后：

$$I_{Q\_rms}^2 = \frac{t_{on}}{T} \cdot I_L^2 + \frac{1}{T} \int_0^{t_{on}} i_{ac}^2(t) \, dt$$

交流分量在 $t_{on}$ 内的 RMS 计算与 2.1 节类似，结果为 $\Delta i_L^2 / 12$。因此：

$$I_{Q\_rms}^2 = D \cdot I_L^2 + D \cdot \frac{\Delta i_L^2}{12} = D \cdot \left(I_L^2 + \frac{\Delta i_L^2}{12}\right)$$

括号内恰好就是电感电流有效值的平方 $I_{L\_rms}^2$：

$$\boxed{I_{Q\_rms} = \sqrt{D} \cdot I_{L\_rms}}$$

> **通用结论：** 若一个波形在整个周期内的有效值为 $I_{rms}$，现截取其 $D \cdot T$ 时间段（其余时间为零），则截取后波形的有效值为 $\sqrt{D} \cdot I_{rms}$。
>
> 对比平均值：截取后波形的**平均值**为 $D \cdot I_{avg}$（线性关系），而**有效值**为 $\sqrt{D} \cdot I_{rms}$（平方根关系）。

### 2.4 两种方法的等价性验证

方法一给出：

$$I_{Q\_rms}^2 = D \cdot \frac{I_{L1}^2 + I_{L1} \cdot I_{L2} + I_{L2}^2}{3}$$

方法二给出：

$$I_{Q\_rms}^2 = D \cdot \left(I_L^2 + \frac{\Delta i_L^2}{12}\right)$$

代入 $I_L = \dfrac{I_{L1} + I_{L2}}{2}$，$\Delta i_L = I_{L2} - I_{L1}$：

$$\begin{aligned}
I_L^2 + \frac{\Delta i_L^2}{12}
&= \frac{(I_{L1} + I_{L2})^2}{4} + \frac{(I_{L2} - I_{L1})^2}{12} \\[4pt]
&= \frac{3(I_{L1}^2 + 2I_{L1}I_{L2} + I_{L2}^2) + (I_{L1}^2 - 2I_{L1}I_{L2} + I_{L2}^2)}{12} \\[4pt]
&= \frac{4I_{L1}^2 + 4I_{L1}I_{L2} + 4I_{L2}^2}{12} \\[4pt]
&= \frac{I_{L1}^2 + I_{L1} \cdot I_{L2} + I_{L2}^2}{3}
\end{aligned}$$

两种方法**完全等价**，殊途同归。

> 验算的意义：在工程中，用两种独立的方法计算同一结果，若一致则增强了结论的可靠性。若不一致，则至少有一种方法存在错误。

## 三、两种方法的对比与选择建议

| | 方法一（直接积分） | 方法二（DC+AC 分解） |
|---|---|---|
| **输入** | $I_{L1}$, $I_{L2}$（谷值、峰值） | $I_L$（直流分量）, $\Delta i_L$（纹波峰峰值） |
| **公式** | $\sqrt{D \cdot \frac{I_{L1}^2 + I_{L1}I_{L2} + I_{L2}^2}{3}}$ | $\sqrt{D} \cdot I_{L\_rms}$ |
| **物理含义** | 纯数学结果 | 分离直流损耗和交流损耗 |
| **适用场景** | 已知波形的两个端点值 | 需要分析 DC/AC 各自损耗占比 |

> **推荐使用方法二**，因为它将**直流损耗**和**交流损耗**分开，便于效率优化——哪个损耗大就针对性地优化哪个。
>
> - $I_{dc}^2 \cdot R_{ds(on)}$ → 导通损耗（直流）
> - $I_{ac\_rms}^2 \cdot R_{ds(on)}$ → 纹波附加损耗（交流）
>
> 方法一可以作为一种**快速验算**的手段。

|---|---|---|
