# 第三章 Buck电路DCM原理 第一节：BCM工作过程

## 一、背景回顾

前面章节讨论的都是 Buck 电路在 **CCM（Continuous Conduction Mode，连续导通模式）** 下的工作过程和参数设计。本章开始讨论 **DCM（Discontinuous Conduction Mode，非连续导通模式）**。

假设已经设计好一个 Buck 变换器（电感 $L$、开关频率 $f_s$、占空比 $D$ 等参数均已固定），它在**满载**情况下工作于 CCM 模式。

但实际工作中，负载 $R_0$ 是会变动的——它不可能永远在满载工作。

---

## 二、进入 BCM 的两种方式

**BCM（Boundary Conduction Mode，临界导通模式）** 是 CCM 与 DCM 之间的临界状态。进入 BCM 有两条路径：

### 2.1 方式一：增大负载电阻 $R_0$（减小负载功率）

当负载功率下降时，$R_0$ 增大。由于输出电压 $U_o$ 恒定：

$$I_o = \frac{U_o}{R_0} \quad \Rightarrow \quad R_0 \uparrow \ \Rightarrow \ I_o \downarrow$$

Buck 电路中电感电流平均值等于负载电流：

$$I_L = I_o$$

因此 $I_o$ 下降意味着 $I_L$ 下降。

**关键观察：电感电流纹波 $\Delta i_L$ 几乎不变。**

在 CCM 模式下：
$$\Delta i_L = \frac{U_o}{L} \cdot (1-D) \cdot T$$

$U_o$、$L$ 不变，CCM 模式下占空比 $D = U_o / U_{in}$ 与负载关系很弱（几乎不变），因此 $\Delta i_L$ **基本不变**。

于是，电感电流波形整体向下平移（平均值下降，纹波保持不变）：

```
i_L
^
|  i_LP ──── +--+           +--+            +--+
|           /    \         /    \          /    \
|          /      \       /      \        /      \
|         /        \     /        \  →   /        \
|  I_L1──+----------+---+----------+──  +----------+──── I_o ↓
|       /            \  /           \   /           \
|      /              \/             \ /             \
|     /               /              V               \
|    /               /                \               \
+--+---------------+--+----------------+---------------+──> t
   |<--- t_on --->|<--- t_off ----->|
   |<------------- T -------------->|

   满载 CCM          负载减轻              负载继续减轻
   (R0 较小)        (R1 > R0)            (R2 > R1)
                                           ↑ 谷底刚好触碰零
                                           此时进入 BCM
```

当电感电流谷底刚好**触碰零点**时，即进入**临界模式（BCM）**。此时的负载电阻称为**临界电阻**。

**临界电阻公式：**

在临界点，$I_o = \dfrac{1}{2} \Delta i_L$：

$$\begin{aligned}
I_o &= \frac{1}{2} \cdot \frac{U_o}{L} \cdot (1-D) \cdot T \\[4pt]
\frac{U_o}{R_{crit}} &= \frac{U_o \cdot (1-D) \cdot T}{2L}
\end{aligned}$$

$$\boxed{R_{crit} = \frac{2L}{(1-D) \cdot T}}$$

> 当 $R_0 > R_{crit}$ 时，电路进入 DCM 模式。

### 2.2 方式二：减小电感 $L$（满载不变）

保持满载（$I_o$ 不变），减小电感 $L$。

由纹波公式：
$$\Delta i_L = \frac{U_o}{L} \cdot (1-D) \cdot T$$

$L \downarrow \ \Rightarrow \ \Delta i_L \uparrow$ —— 纹波增大。

但电感电流平均值 $I_L = I_o$ 不变（负载没变），所以：

```
i_L                    i_L                    i_L
^                      ^                      ^
|     +--+             |       +--+           |         +--+
|    /    \            |      /    \          |        /    \
|   /      \           |     /      \         |       /      \
|  /        \          |    /        \        |      /        \
| +----------+--       |   +----------+--     |     +----------+--
|/            \        |  /            \      |    /            \
|              \       | /              \     |   /              \
|               \      |/                \    |  /                \
+--+-------------+--   +--+--------------+--  +--+----------------+--
   L1 (原始)             L2 < L1               L3 = L_crit
   中点 = I_o            中点 = I_o             谷底刚好触碰零
                                                 ↑ 此时进入 BCM
```

当谷底刚好触碰零点时，对应的电感值称为**临界电感**：

$$\boxed{L_{crit} = \frac{R_0 \cdot (1-D) \cdot T}{2}}$$

> 当 $L < L_{crit}$ 时，电路进入 DCM 模式。

---

## 三、BCM 模式的关键特性

一旦进入临界模式，以下关系依然成立：

$$I_o = I_L = \frac{1}{2} \cdot I_{LP}$$

其中 $I_{LP}$ 是电感峰值电流（此时谷底 $I_{L1} = 0$）。

此时纹波 $\Delta i_L = I_{LP} - 0 = I_{LP}$，因此：

$$\Delta i_L = 2 \cdot I_o$$

电流纹波率：
$$\boxed{r = \frac{\Delta i_L}{I_L} = 2}$$

> **重要结论：当 $r = 2$ 时，电感进入临界模式（BCM）。**
>
> 这个结论不单单对 Buck 电路成立，对**所有基础拓扑**都成立。

---

## 四、BCM 的定位

在临界点上，变换器工作于 BCM 模式。它既可以看成 CCM 的边界，也可以看成 DCM 的边界——**两者是无缝衔接的**。

| 模式 | 条件 | 电感电流特征 |
|------|------|-------------|
| **CCM** | $R_0 < R_{crit}$ 且 $L > L_{crit}$ | 电感电流始终 > 0，$r < 2$ |
| **BCM** | $R_0 = R_{crit}$ 或 $L = L_{crit}$ | 谷底刚好触碰零，$r = 2$ |
| **DCM** | $R_0 > R_{crit}$ 或 $L < L_{crit}$ | 电感电流出现零区间，$r > 2$ |

---

## 五、公式汇总

| 参数 | 公式 | 说明 |
|------|------|------|
| 临界电阻 | $R_{crit} = \dfrac{2L}{(1-D) \cdot T}$ | $R_0 > R_{crit}$ 进入 DCM |
| 临界电感 | $L_{crit} = \dfrac{R_0 \cdot (1-D) \cdot T}{2}$ | $L < L_{crit}$ 进入 DCM |
| BCM 纹波率 | $r = 2$ | 所有基础拓扑通用 |
| BCM 峰值电流 | $I_{LP} = 2 \cdot I_o$ | 谷底为零时的特例 |
