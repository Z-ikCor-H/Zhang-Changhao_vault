# 第三章 Buck DCM原理 第二节：DCM工作过程

上一节分析了临界条件。本节讨论越过临界点后的DCM工作过程：**电感电流波形如何变化？为什么出现 $Q$ 和 $D$ 同时关断的死区时间？二极管阻断负电流对波形有什么约束？零电流开关带来哪些优势和代价？** 分析DCM的三个工作模态和电流波形的变化规律。

---

## 一、从BCM进入DCM：第三模态的出现

### 1.1 负载继续减轻（$R_0 \uparrow > R_{crit}$）

在BCM基础上，若负载电阻继续增大（输出功率继续下降）：

- 按CCM的规律，电感电流波形应继续向下平移
- 这意味着电感电流会出现**负值区域**

但在实际Buck电路中，电感电流**不能为负**：

![[3.1-1.png|516]]

- **二极管 D**：电流若反向（从阴极流向阳极），二极管截止 → 此路不通
- **开关管 Q**：虽然MOS管体二极管可反向导通，但 Buck 电路中 $U_{in} > U_o$，反向电流被输入电压阻挡

> 如果将二极管替换为同步整流MOS管（双向导通），电流确实可以为负，进入**强迫CCM（FCCM）**模式。本节不讨论这种情况。

### 1.2 斜率的继承

当仅改变负载 $R_0$ 时：
- $U_{in}$、$U_o$、$L$ 均不变
- 上升斜率：$\dfrac{U_{in} - U_o}{L}$ 不变
- 下降斜率：$\dfrac{U_o}{L}$ 不变

因此DCM电流波形与BCM波形具有**相同的斜率**，只是峰值降低、底边变窄。

## 二、DCM的三个工作模态

DCM模式下，Buck电路每个开关周期包含三个阶段：

| 模态 | 时段 | Q | D | 电感电流 | 负载供电 |
|---|---|---|---|---|---|
| **模态一** | $t_{on} = DT$ | 导通 | 截止 | 线性上升（斜率 $\frac{U_{in}-U_o}{L}$） | 电源+电感 |
| **模态二** | $t_{off} = D_1 T$ | 截止 | 导通 | 线性下降（斜率 $-\frac{U_o}{L}$） | 电感释放 |
| **模态三** | $T_D = T - t_{on} - t_{off}$ | 截止 | 截止 | **为零** | **仅靠输出电容** |

> 模态三是DCM**独有**的——电感电流已降为零，开关管和二极管均关断，负载完全由输出电容 $C_o$ 维持供电。

![[3.2-3.png]]

### 关键参数定义

- $D$：开关管导通占空比（$t_{on}/T$）
- $D_1$：二极管导通占空比（$t_{off}/T$）
- $T_D$：断续时间（死区时间），$D + D_1 < 1$

## 三、电流波形的变化分析

### 3.1 负载增大 $R_0$ 的情况

电感电流平均值必须等于负载电流 $I_o$：

$$I_L = I_o = \frac{U_o}{R_0}$$

$R_0 \uparrow$ → $I_o \downarrow$ → 需要三角形面积减小。

在斜率不变的前提下：
- 峰值 $I_{LP}$ 降低
- 导通时间之和 $t_{on} + t_{off}$ 缩短
- 出现零电流区间 $T_D$

> **直观理解：** 斜率相同的两个三角形，底边越短、高度越低，面积越小。由于峰值和底边由相同斜率耦合，二者同步减小。

### 3.2 电感减小 $L$ 的情况（$L < L_{crit}$）

$L$ 减小会导致**斜率发生变化**：

- 上升斜率 $\frac{U_{in}-U_o}{L}$ **增大**
- 下降斜率 $\frac{U_o}{L}$ **增大**

此时波形变化较复杂——峰值 $I_{LP}$ 是升高还是降低？占空比如何调整？

> 需要通过数学推导来分析，这将在下一节（DCM增益分析）中完成。分析完成后，再回过头来理解其物理意义。

## 四、DCM模式的特点

### 4.1 优势

**① 二极管的零电流关断（ZCS）**

![[3.2-5.png]]

- 二极管电流在模态二结束时**自然降为零**后才关断
- 无反向恢复问题（类似LLC谐振变换器）
- 反向恢复损耗几乎为零

**② 开关管的零电流开通**

- 开关管在模态三结束时开通，此时电感电流为零
- 开通瞬间无电流 → 开通损耗显著降低

> 注意：开关管关断时电感电流为峰值 $I_{LP}$，**并非零电流关断**。但与CCM相比，由于无二极管反向恢复尖峰叠加，关断损耗也有所改善。

### 4.2 劣势

**① 峰值电流更高**

![[3.2-6.png]]

在相同 $I_o$（相同输出功率）下：
- DCM三角形底边 $t_{on} + t_{off} < T$（比BCM更短）
- 但面积必须相等（平均值 = $I_o$）
- → 需要更高的峰值 $I_{LP}$

$$I_{LP\_DCM} > I_{LP\_BCM}$$

这意味着开关管和二极管的**电流应力更大**。

**② 电流纹波更大**

- 纹波 $\Delta i_L = I_{LP}$（谷底为零）
- 需要更大容量的输出电容来滤波
- 电容RMS电流增大

**③ 电感体积更小（这是一把双刃剑）**

- $L < L_{crit}$，电感感值减小 → 同磁芯下匝数更少 → 体积更小
- 但电流峰值增大 → 需要更粗的线径或应对更大的磁芯损耗

| 特性 | CCM | DCM |
|---|---|---|
| 二极管关断 | 硬关断，有反向恢复 | **ZCS关断**，无反向恢复 |
| 开关管开通 | 带电流开通 | **零电流开通** |
| 峰值电流（同$I_o$） | 较小 | **较大** |
| 电流纹波 | 较小 | **较大** |
| 输出电容需求 | 较小 | **较大** |
| 电感体积 | 较大 | **较小** |

## 五、本节小结

| 主题 | 核心结论 |
|---|---|
| 第三模态 | DCM独有：$Q$和$D$均截止，$i_L=0$，电容单独供电 |
| 占空比关系 | $D + D_1 < 1$，断续时间 $T_D = (1-D-D_1)T$ |
| 电流不能为负的原因 | 二极管反向截止 + Buck输入电压阻挡 |
| 斜率不变（仅$R_0$变化时） | 上升/下降斜率与CCM相同，峰值随$I_o$同步减小 |
| DCM优势 | ZCS关断（无反向恢复）、零电流开通 |
| DCM劣势 | 峰值电流高、纹波大、电容需求大 |
