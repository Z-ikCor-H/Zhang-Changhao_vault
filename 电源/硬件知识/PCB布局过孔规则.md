● 这是一个非常好的问题，也是电源PCB设计中非常关键的技术。让我从原理到实践给你系统讲一下。

  ---
  一、为什么GND旁边要打许多过孔？

  核心原因有三个：

  1. 降低回流路径阻抗（最关键）

  电流永远是"有去有回"的——信号从IC出来，最终要回到GND。高频电流总是走阻抗最小的路径，也就是电感最小的路径。如果GND回流路
  径绕了一大圈，回路面积大，电感就大，会导致：
  - 地弹（Ground Bounce）：不同位置的GND电位不同，产生噪声
  - EMI辐射严重
  - 信号完整性变差

  在GND焊盘旁边打过孔，就是给回流电流提供一条最短、最低阻抗的回家路。

  2. 散热

  电源芯片（LDO、DC-DC等）发热量不小。GND通常是铜皮，打过孔把热量传导到内层或底层的地平面，相当于多通道散热。很多电源芯
  片的数据手册会明确写"Use multiple thermal vias"。

  3. 电流承载能力

  大电流情况下，一个过孔承载的电流有限。多个过孔并联分担电流，防止单一过孔过热甚至烧断。

  ---
  二、怎么打？具体规则

  放置位置

  - 紧贴GND焊盘：过孔离焊盘越近越好（通常2-5mil），让回流路径最短
  - 环绕式或多排排列：在GND焊盘周围均匀分布
  - 不要打在焊盘上：除非是"盘中孔"（via-in-pad）工艺（需要填平+电镀封孔，成本高），否则不要打在SMD焊盘上，否则焊接时锡膏
  会流入过孔造成虚焊

  典型布局示意

    散热焊盘（GND PAD）
  ┌─────────────────────┐
  │  ○  ○  ○  ○  ○  ○  │    ← 过孔环绕或阵列排布
  │  ○  ○  ○  ○  ○  ○  │
  │  ○  ○  ○  ○  ○  ○  │
  └─────────────────────┘

  间距规则

  - 孔到孔间距（hole-to-hole）：≥ 0.25mm（10mil），取决于PCB厂工艺能力
  - 孔到铜皮（via-to-copper）：≥ 0.15mm（6mil）
  - 过孔与焊盘边缘距离：0.1mm-0.2mm（4-8mil）

  数量怎么定？

  - 一般信号/逻辑IC：2-4个
  - 电源芯片（LDO, DC-DC）：4-16个，看功耗
  - 大电流场景（>5A）：按过孔载流能力反算

  ▎ 一个0.3mm孔径/0.6mm外径的过孔大约能承载0.5-1A，以此估算数量。

  ---
  三、打多大？尺寸选择

  过孔结构

          焊环（Annular Ring）
         ┌─────────────┐
         │  ┌───────┐  │
         │  │ 孔壁  │  │
         │  │ 镀铜  │  │
         │  └───────┘  │
         └─────────────┘
          ← 外径(Drill+2×Ring) →
                ← 孔径(Drill) →

  常用尺寸推荐

  ┌─────────────┬──────────────────────┬──────────────────────┬────────────────┐
  │    场景     │     孔径 (Drill)     │   外径 (Diameter)    │      说明      │
  ├─────────────┼──────────────────────┼──────────────────────┼────────────────┤
  │ 常规GND过孔 │ 0.3mm (12mil)        │ 0.6mm (24mil)        │ 最常用         │
  ├─────────────┼──────────────────────┼──────────────────────┼────────────────┤
  │ 密集区域    │ 0.2mm (8mil)         │ 0.45mm (18mil)       │ 空间受限时     │
  ├─────────────┼──────────────────────┼──────────────────────┼────────────────┤
  │ 大电流      │ 0.4-0.5mm (16-20mil) │ 0.8-1.0mm (32-40mil) │ >2A时考虑      │
  ├─────────────┼──────────────────────┼──────────────────────┼────────────────┤
  │ 散热为主    │ 0.3mm (12mil)        │ 0.6mm-0.8mm          │ 填充或开窗更好 │
  └─────────────┴──────────────────────┴──────────────────────┴────────────────┘

  选型原则

  - 孔径 ≥ PCB板厚的 1/8 ~ 1/6（保证电镀可靠）
  - 焊环宽度 ≥ 0.1mm (4mil)，太小了钻孔偏位容易破盘
  - 1.6mm厚板子最常见打0.3mm孔径/0.6mm外径

  ---
  四、Altium Designer 中的操作方法

  1. 放置过孔：快捷键 P → V，或工具栏点击过孔图标
  2. 设置网络：将过孔网络设为 GND
  3. 快速阵列放置：
    - 选中一个过孔，Ctrl+C / Ctrl+V 复制粘贴
    - 或者用 Tools → Via Stitching 自动添加缝合过孔
  4. 在PCB规则中设定：Design → Rules → Routing Via Style，设定你偏好的过孔尺寸

  ---
  五、进阶技巧

  阻焊开窗（Tenting vs Opening）

  - 盖油（Tented）：过孔被绿油覆盖，常规做法
  - 开窗（Open）：过孔露出，散热更好但可能氧化/短路风险，散热场景可用

  过孔塞孔（Via Plugging / Filling）

  - 先用树脂填平，再盖油，防止锡膏流失和空洞，BGA下方必须塞孔

  电源PCB特别建议

  对于你这个电源设计项目：
  - DC-DC芯片的GND散热焊盘区域，打 4-9个 0.3/0.6mm过孔
  - 输入/输出电容的GND端每颗打 2-3个 过孔直接到地层
  - 大电流路径上的过孔偏多不偏少

  ---
  总结一句话：过孔就是给电流修"快速通道"，离GND越近、数量越多、分布越均匀，回路电感越小，板子越稳。
  对电源板来说，过孔不是"有就行"，而是"多多益善但要合理"。

❯ 怎么隐藏元件旁边的标号

  Thought for 2s (ctrl+o to expand)

● 在 Altium Designer 中隐藏元件标号（位号，如 R1、C1、U1），有以下几种方法：

  ---
  方法一：批量隐藏所有标号（最常用）

  1. 在 PCB 编辑器中，右键任意一个标号文字（如 R1）
  2. 选择 Find Similar Objects...
  3. 在弹出的对话框中设置：
    - Object Kind 设为 Text → Same
    - String Type 设为 Designator → Same
    - 点击 OK
  4. 在弹出的 PCB Inspector 面板中，找到 Hide 选项，打勾即可
  5. 按 Shift+C 取消选中

  ---
  方法二：单个隐藏

  1. 直接点击选中某个元件旁边的标号文字
  2. 在右侧 Properties 面板中找到 Hide 选项，打勾

  ---
  方法三：控制标号显示/隐藏层

  1. 按快捷键 L（或菜单 View → Panels → View Configuration）
  2. 在 Layers & Colors 面板中
  3. 点击 Show/Hide 标签
  4. 找到 Designators 行，切换为隐藏模式

  ▎ 或者直接按 Shift+S 切换单层模式查看，不改变实际显示设置。

  ---
  方法四：在原理图中批量控制（从源头）

  1. 在原理图中，右键任一元件 → Find Similar Objects...
  2. 设置 Object Kind = Part → Same，点 OK
  3. 在 SCH Inspector 面板中，找到 Show Designator（或类似字段），设为 False
  4. 重新同步到 PCB（Design → Update PCB Document）

  ---
  推荐方法一和方法三，方法三最安全，不改动文件属性，只是在显示层面开关。
